- Coffee-script
- Grunt

## Usage

[mffr][] is required.

    $ git clone https://github.com/cybertk/npm-scaffolding.git
    $ cd npm-scaffolding
    $ mv README.app.md README.md
    $ mffr npm-scaffolding <your_package_name>

[mffr]: https://github.com/cybertk/mffr
